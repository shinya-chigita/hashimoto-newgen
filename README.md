# はしもと新時代 公式サイト（ドラフト）

添付ビラPDFをもとに作成したシングルページサイトです。

## 構成

- `index.html` - 本体
- `styles.css` - スタイル
- `assets/flyer.pdf` - 元資料PDF

## ローカルで確認

```bash
cd hashimoto-newgen
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## 注意

- PDFの一部テキストはOCR判読が難しかったため、要確認のプレースホルダ表記を含みます。
- 連絡先メールは仮置き（`info@example.jp`）です。正式情報に差し替えてください。
